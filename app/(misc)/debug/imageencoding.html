Step 2 Resizing and Standardization

The image is automatically resized to a 30x30 pixel grid, creating a standardized input size.

Step 3 RGB Value Extraction

Each pixel's RGB color values (0-255) are extracted and displayed in the grid below.

Step 4 Grayscale Conversion

RGB values are normalized to a 0-1 range and converted to grayscale (a common simplification for neural networks). This can be visualized as a grid or reconstructed image.

Grayscale image reconstructed from the normalized values (0 = Black, 1 = White).
